**Програма для резервування столиків**
При запуску програми (`Main.java`) користувачеві доступне:

1. **Резервування столика:**
   * Користувач вводить бажану дату (у межах `14` днів наперед).
   * Система показує список усіх столиків та інтервали їхньої доступності.
   * Користувач обирає столик, час початку та тривалість.
   * Програма перевіряє сумісність замовлення з графіком роботи закладу (з 10:00 до 23:00). У разі успіху створюється запис зі статусом `PENDING`.
2. **Перегляд існуючих бронювань:**
   * Фільтрація та перегляд списку замовлень за конкретною датою.
   * Перегляд історії та поточного статусу бронювань по ID конкретного столика.
3. **Вихід з автозбереженням:**
   * Завершення сесії користувача із гарантованим збереженням усіх нових записів у файли.
     
Дані записуються в текстові файли порядково. Приклад структури:

* **tables.txt:** `123|6|CHAIR|Near window|Near AC`
* **reservations.txt:** `PENDING|2026-04-29T14:30|6|123|2|client@email.com|+0630000000`

6. Large Data Processing with Streams

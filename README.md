Програма для автоматизації процесу бронювання столиків у ресторані, розроблена в рамках курсового проекту.
Проект дозволяє керувати базою даних столів та бронювань у консольному режимі:

Управління столиками: зберігання інформації про ідентифікатор, кількість місць, тип сидінь (стільці, лавки, дивани) та розташування.
Система бронювання: можливість створення резервації з вибором дати, часу та кількості осіб.
Два сценарії пошуку:Вибір дати перегляд списку вільних столів.Вибір конкретного столика перегляд доступних дат для нього.
Валідація даних: перевірка коректності введення контактної інформації (телефон/email).
Фільтрація через Stream API: вибірка вільних столів на сьогодні або на конкретний час.
